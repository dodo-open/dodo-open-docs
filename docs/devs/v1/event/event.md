# 事件主体

## 接收方式

开发者当前可通过[WebSocket](./websocket.md)接收各类事件

## 事件主体

EventSubject

|字段|类型|说明|
|:---------------|:-----|:---------------|
|type|int|数据类型，0：[业务数据](#业务数据)|
|data|object|数据内容|

## 业务数据

EventSubjectDataBusiness

|字段|类型|说明|
|:---------------|:-----|:---------------|
|eventId|string|事件ID|
|eventType|string|[事件类型](#事件类型)|
|eventBody|object|事件内容|
|timestamp|long|发送时间戳|
